# 🌐 Personal Developer Portfolio

A responsive personal portfolio website built with React to showcase my projects, technical skills and professional background.

This project focuses on clean UI design, modular component structure and responsive layout.

---

## 🛠 Technologies

- React
- JavaScript (ES6+)
- CSS / SCSS
- Responsive Design

---

## ✨ Features

- 🏠 Hero section with introduction
- 👨‍💻 About section
- 📂 Projects showcase
- 🛠 Skills section
- 📩 Contact section
- 📱 Fully responsive layout

---

## 🧠 The Process

The portfolio was built with a modular component-based architecture:

- Each section is structured as a reusable component.
- Layout is designed to be clean, minimal and easy to navigate.
- The project emphasizes readability, spacing and consistent UI structure.
- Focus was placed on responsive behavior across different screen sizes.

The goal was to create a simple yet professional web presence that reflects my technical profile.

---

## 📚 What I Learned

- Structuring personal branding through code
- Designing clean and readable layouts
- Creating reusable section-based components
- Improving UI consistency and spacing
- Organizing projects in a presentable way

## ▶️ Running the Project

Clone the repository:

```bash
git clone https://github.com/AlexFranciscoDev/portfolio.git
```

Install dependencies:

```
npm install
```

Run locally:

```
npm start
```

App available at:

http://localhost:3000
